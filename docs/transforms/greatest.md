##### Function: `greatest`
```
  return the largest value among its arguments (ignoring null)

Examples:

  greatest(5, 4, 3, 2, 1)
  -- Result: "5"

  greatest(4, 1, null, 3, 6)
  -- Result: "6"

  greatest(null, null, null)
  -- Result: null




```

###### Signatures
    a -> a
