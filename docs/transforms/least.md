##### Function: `least`
```
  return the smallest value among its arguments (ignoring null)

Examples:

  least(5, 4, 3, 2, 1)
  -- Result: "1"

  least(4, 1, null, 3, 6)
  -- Result: "1"

  least(null, null, null)
  -- Result: null




```
